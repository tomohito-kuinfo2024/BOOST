Begin-Oriented Operating SysTem

The Operating System for Getting Started.

## 概要
行動開始時の心理的ハードルを下げるため、入力内容に応じて複数の行動案と危険度を提示するWebアプリです。

## 主な機能
- 気分・使える時間の入力内容に応じた3つの提案生成
- 4段階の危険度判定
- モチベーションアップのためのご褒美提案

## 開発で重視した点
実装そのものだけでなく、利用者が行動に移しやすい提案粒度や評価導線を意識して設計しました。またモチベーションを湧かせるために楽しみな予定を登録して表示させたり、今日のご褒美を設定・提案できる機能を実装しました。
